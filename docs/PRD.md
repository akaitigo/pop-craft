# POP-Craft — Product Requirements Document

## 概要

小売店舗のPOP（店頭販促物）をテンプレートから即時生成するPOPデザインツール。
「本日のおすすめ」「期間限定」「店長イチオシ」など日本の小売現場で頻出するPOPパターンをテンプレート化し、商品名・価格・キャッチコピーを入力するだけで印刷可能なPOPを生成する。

## 背景・課題

- 小売現場のPOP作成はExcel・手書きに依存し、属人的かつ非効率
- デザインツール（Canva等）は汎用すぎて小売POP特化のテンプレートがない
- 日本語の手書き風・筆文字フォントとの組み合わせが手間
- 用紙サイズが日本のJIS規格（A4/A5/名刺サイズ）に最適化されていない

## ターゲットユーザー

- スーパー・ドラッグストア・書店の店舗スタッフ
- POP作成を担当するバイト・パート従業員
- デザイン経験のない店長・売場担当者

## MVP機能

### F1: 業態別POPテンプレート選択・商品情報入力
- 業態カテゴリ: スーパー / ドラッグストア / 書店
- テンプレートパターン: 本日のおすすめ / 期間限定 / 店長イチオシ / 新商品 / セール
- 入力フィールド: 商品名、価格（税込/税抜）、キャッチコピー、説明文（任意）
- **受け入れ条件**: 業態選択→テンプレート選択→商品情報入力の3ステップでPOPが生成される

### F2: デザインカスタマイズ
- フォント選択: ゴシック / 明朝 / 手書き風 / 筆文字（4種以上）
- カラーパレット: 業態別推奨カラー + カスタムカラー
- レイアウト微調整: 文字サイズ・配置の変更
- Canvas APIによるリアルタイムプレビュー
- **受け入れ条件**: フォント・カラー変更が即座にプレビューに反映される

### F3: PDF出力・印刷プレビュー
- 用紙サイズ: A4 / A5 / 名刺サイズ（91mm × 55mm）
- PDF生成（Go側でサーバーサイド生成）
- 印刷プレビュー表示
- ダウンロードボタン
- **受け入れ条件**: 選択した用紙サイズで正確なPDFが生成・ダウンロードできる

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| Frontend | TypeScript / Next.js 15 (App Router) |
| POP描画 | Canvas API |
| Backend API | Go (net/http + chi router) |
| PDF生成 | Go (go-fpdf) |
| Database | PostgreSQL |
| インフラ | GCP Cloud Run（将来） |

## 非機能要件

- PDF生成: 3秒以内
- フロントエンドプレビュー: 200ms以内で描画更新
- 日本語フォント: Web Font + PDF埋め込み対応

## スコープ外（MVP）

- ユーザー認証・マルチテナント
- POP履歴の永続保存（ローカルストレージのみ）
- カスタムテンプレート作成機能
- 画像・ロゴのアップロード

## 成功指標

- 3ステップ以内でPOP生成完了
- PDF出力が全対応用紙サイズで正確
- テンプレート3業態 × 5パターン = 15種以上
